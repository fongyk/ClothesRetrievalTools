# ClothesRetrievalTools

## Data organization

```
category
└── pair_id
    └── style
        ├── shop
        │   ├── xxxx.jpg
        │   └── xxxx.jpg
        └── user
            ├── xxxx.jpg
            └── xxxx.jpg
```

Query images are from users (consumers), gallery images are from shops.

Images iif with same pair_id and style are positive pairs.
